I find Web Audio API interesting and can be used for various user-engaging projects. Web API provides a powerful and versatile system for controlling audio on the Web, allowing developers to choose audio sources, add effects to audio, create audio visualizations, apply spatial effects (such as panning) and much more.
The most interesting about this API is that it can extractrequency, waveform, and other data from the audio source, so it makes the visualization of a certian audio possible. Bringing countless possibilities for application in Drawing on the Web. It can also corporate with user's microphone in order to achieve the purpose of interacting with users in real time.
